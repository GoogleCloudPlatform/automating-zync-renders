# Images directory
This directory will contain the final, rendered images generated by ZYNC. Once rendered, images are downloaded by a running ZYNC client.
Renders for each object will be in a subdirectory named after the part. For example:
```
[WORKING-DIRECTORY]/
|____images/
| |____part01/
| | |____part01.0001.jpg
| | |____part01.0002.jpg
| | |____part01.0003.jpg
| | |____etc...
| |____part02/
| | |____part02.0001.jpg
| | |____part02.0002.jpg
| | |____part02.0003.jpg
| | |____etc...
| |____part03/
| | |____part03.0001.jpg
| | |____part03.0002.jpg
| | |____part03.0003.jpg
| | |____etc...
```
